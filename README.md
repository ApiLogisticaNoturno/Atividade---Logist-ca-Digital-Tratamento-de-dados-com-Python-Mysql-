# 📦 Projeto Logística - Banco de Dados MySQL

Este projeto simula um sistema simples de gestão logística utilizando MySQL e Python para análise de dados.

## 📊 Objetivo

Criar um banco de dados logístico contendo:

- Clientes
- Produtos
- Transportadoras
- Pedidos
- Itens de pedido

Permitindo consultas e análises de dados logísticos.

## 🗄 Estrutura do Banco de Dados

Tabelas criadas:

- Clientes
- Produtos
- Transportadoras
- Pedidos
- ItensPedido

Relacionamentos:

Pedidos → Clientes  
Pedidos → Transportadoras  
ItensPedido → Pedidos  
ItensPedido → Produtos

## 🛠 Tecnologias utilizadas

- MySQL
- Python
- Pandas
- Visual Studio Code


## 📈 Possibilidades de análise

Com esse banco de dados é possível analisar:

- volume de pedidos
- produtos mais vendidos
- faturamento
- desempenho de transportadoras
